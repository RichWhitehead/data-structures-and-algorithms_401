# Repeated Word

## Author: Richard Whitehead

## Challenge

Find the first repeated word in a book.

## Approach & Efficiency

Time Big O(l) where l is length of the string in words. Space Big O(l)

## Input: "Once upon a time, there was a brave princess who..."

## Output: 

"The"

# Input: 

"a b c, d"

## Output: 

None
